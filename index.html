小时候的记忆

（谨以此文，献给终身劳作不辞辛苦的父亲和母亲）

​ 很多时候都想写点什么，来讲述自己的经历，可是自己又像普通蚂蚁中的一只，没有任何可以记载的意义。

​ 记得小时候很喜欢朱自清的散文，讲述了自己的父亲，那一辈人的心酸和艰辛，也喜欢路遥的平凡的世界，读着读着就哭了。那里面平凡的人因为和他们有近似的遭遇而感慨万千。

上传本地博客教程

写于2019年12月14日 星期六

  1. 在 Blog_HBYang 文件夹上右键 选择 git bash here
  2. 在 控制台输入 hexo new test
  3. 此时会在 G:\Blog_HBYang\source_posts 目录下生成一个 test.md的文件，此时用typora进行修改或者创作，导出到本文件夹 html的文件
  4. 使用 hexo g 的命令，生成静态文件
  5. 使用hexo d 将其部署到GitHub上
  6. 完成

注意 如果采用typora自带的 图片插入方法，网页端是无法显示图片的，而应该采用以下方式
